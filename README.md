# 2D-Projectile-Motion-Visualization-Application-with-Linear-and-Quadratic-Drag
2D Projectile Motion Simulator built with Python and Tkinter. Compares three trajectory models — ideal (no drag), linear drag, and quadratic drag (LAT approximation) — with real-time animation, color-coded trails, and a results summary showing range, max height, and flight time for each model.
